How to use
----------
1) Replace placeholder assets in /assets with your product photos:
   - 001.jpg, 002.jpg, 003.jpg, ... (keep the numbers matching your sheet)
   - hero.jpg (homepage hero banner)
   - placeholder.jpg (fallback image, optional)

2) Open `app.js` and update:
   - WA_NUMBER with your real WhatsApp number (e.g., 919876543210)
   - The `products` array: name, sku, price, image path (e.g., 'assets/007.jpg')

3) Open `index.html` directly in your browser to preview.

Deploy (Free options)
---------------------
- GitHub Pages: create a public repo, upload these files at the root, then enable Pages.
- Netlify: drag-and-drop the folder at https://app.netlify.com/drop
- Vercel: import the folder as a static site.

Branding
--------
- Update text content in index.html (hero, about, contact).
- Add your Instagram and email.
